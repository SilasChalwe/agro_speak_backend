API Information

Available endpoints and their usage

POST

/translate

Translate text between English and Bemba

Response:
{
  "originalText": "Hello",
  "sourceLang": "English",
  "targetLang": "Bemba",
  "translation": "Muli shani",
  "processingTime": "1.23s"
}
POST

/tts

Convert text to speech (returns WAV file)

Returns:
Audio WAV file for direct playback
POST

/tts-base64

Convert text to speech (returns base64 JSON)

Response:
{
  "text": "Muli shani",
  "audio": "base64encodedstring...",
  "audioFormat": "audio/wav",
  "sampleRate": 24000,
  "generationTime": "1.45s"
}
POST

/translate-and-speak

Optimized combined translation + TTS

Response:
{
  "originalText": "Hello",
  "sourceLang": "English",
  "targetLang": "Bemba",
  "translatedText": "Muli shani",
  "translationTime": "1.23s",
  "audioAvailable": true,
  "audio": "base64encodedstring...",
  "totalTime": "2.68s"
}
POST

/translate-and-speak-parallel

Advanced parallel processing for maximum performance

Response:
{
  "originalText": "Hello",
  "sourceLang": "English",
  "targetLang": "Bemba",
  "translatedText": "Muli shani",
  "translationTime": "1.15s",
  "performance": "advanced_parallel",
  "audioAvailable": true,
  "audio": "base64encodedstring...",
  "totalTime": "1.89s"
}
GET

/health
